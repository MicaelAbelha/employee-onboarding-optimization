Employee Onboarding System Optimization

> Agile project: Reduced employee onboarding time by 65% (10 days → 3.5 days) through Jira automation and process redesign

📊 Project Summary

| Attribute | Details |
| **Organization**| Tech startup (100 employees) |
| **Role** | Project Manager |
| **Duration** | 10 weeks (Jan-Mar 2024) |
| **Methodology** | Agile/Scrum (5 sprints × 2 weeks) |
| **Tools** | Jira, Confluence, Google Sheets |

🎯 Challenge

- New hires waited **10-14 days** for full setup
- Only **23% positive feedback** on onboarding
- **15 touchpoints** across 5 departments, no tracking
- Manual email coordination = chaos

💡 Solution

**Automated workflow using Jira:**
- Manager creates ticket → auto-assigns subtasks (IT, Finance, Facilities)
- Automation rules: emails, reminders, notifications
- Confluence guides for self-service onboarding

 📈 Results

| Metric | Before | After | Change |
|--------|--------|-------|--------|
| Onboarding time | 10 days | 3.5 days | ⬇️ 65% |
| Satisfaction | 23% | 87% | ⬆️ 64pts |
| Task completion | 78% | 100% | ⬆️ 22% |
| Time saved | - | 15h/month | HR+IT |

**Business Impact:** €18,800/year (retention improvement + time savings)
